# Calculadora "Simples" com Operações Avançadas

## 📋 Descrição do Projeto

Uma calculadora em Python com múltiplas funcionalidades, desenvolvida como exercício de aprendizado.

**Desenvolvido por:** Albertino Lopes  
**Com mentoria de:** GitHub Copilot (Inteligência Artificial)

---

## ✨ Funcionalidades

1. **Soma** - Soma múltiplos números
2. **Subtração** - Subtrai múltiplos números
3. **Multiplicação** - Multiplica números com validação para zero
4. **Divisão** - Divide números com proteção contra divisão por zero
5. **Percentagem** - 3 tipos de cálculos percentuais
6. **Raiz Quadrada** - Com validação para números negativos
7. **Raiz Cúbica** - Cálculo de raiz cúbica
8. **Elevação a Potência** - Simples ou múltipla
9. **Operações com Pi** - Cálculos geométricos (círculo, esfera, cilindro)

## 📝 Como Usar

```bash

```

2. Escolha uma operação (1-9)
3. Siga as instruções na tela
4. Visualize o resultado
